## 📌 About The Project

**Nivio** is a modern and responsive E-Commerce website built using HTML, CSS, and JavaScript.  
It focuses on clean UI design, smooth user experience, and interactive frontend functionality.

---

## ✨ Features

- 🛒 Add to Cart functionality  
- 📱 Fully Responsive Design  
- 🏷️ Product Listing & Product Detail Pages  
- 🎨 Modern UI Layout  
- ⚡ Smooth Animations  
- 📂 Organized Folder Structure  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
